# ✨ 幸運抽籤與分組神器 (Lucky Draw & Grouping Tool) -- Cyberpunk Edition

## 📖 專案簡介
這是一個現代化、極具未來感的網頁工具，專為需要快速進行名單抽籤或隨機分組的場合設計。
全新升級為 **Cyberpunk Neon Glassmorphism** 風格，結合深邃的星空背景與動態霓虹光效，為您的活動增添強烈的視覺衝擊。

## 🚀 功能特色
- **🤖 Cyberpunk 未來風格**：深色星空背景、動態漂浮光球、高對比的霓虹配色，打造沉浸式體驗。
- **💎 玻璃擬態 (Glassmorphism)**：全介面採用半透明磨砂玻璃質感，搭配發光邊框，科技感十足。
- **📱 完全響應式設計 (RWD)**：重新設計的穩固佈局，從手機到 4K 螢幕都能完美顯示。
- **🎲 雙核心功能**：
  - **幸運抽籤**：大螢幕等級的抽獎動畫，支援單人及多人同時抽出，結果具有震撼的發光特效。
  - **隨機分組**：支援「按組數分」或「按每組人數分」，結果卡片清晰易讀。
- **🔒 純前端運作**：所有資料與邏輯皆在瀏覽器端極速處理，確保隱私。

## 🛠 技術堆疊
- **HTML5**：語意化結構。
- **CSS3**：
  - 使用 `:root` 變數管理霓虹色系。
  - 大量運用 `backdrop-filter` 實現玻璃特效。
  - CSS Animation 實現流暢的進場與互動動畫。
- **JavaScript (ES6+)**：模組化邏輯，包含陣列洗牌算法 (Fisher-Yates Shuffle) 與 Canvas 紙花特效。

## 📂 檔案結構
- **`index.html`**：網頁骨架，包含動態背景結構。
- **`style.css`**：視覺核心，定義了所有的 Cyberpunk 配色、玻璃材質與 RWD 斷點。
  - 若需修改主題色，請調整 `:root` 中的 `--primary`, `--secondary`, `--accent` 變數。
- **`script.js`**：大腦中樞，處理資料流、抽籤邏輯與 Canvas 紙花繪製。

## 🎨 風格指南
本專案目前的視覺主題設定為：
- **背景**：深靛藍 (#0f172a) + 動態光球
- **主色 (Primary)**：霓虹青 (#0bf4f3)
- **次色 (Secondary)**：深紫 (#7000ff)
- **強調色 (Accent)**：霓虹粉 (#f72585)
- **文字**：純白發光樣式，確保在深色背景下的極致可讀性。
